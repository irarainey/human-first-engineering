- Initial release of the Human-First Engineering manifesto.
- Initial release of the five-pillar framework: Think First, Own the Output, Grow Skills, Use AI Intelligently, Trust but Verify.
- Toolkit:
  - Implementation guide with a 10-step rollout.
  - [Practices](toolkit/practices.md) document — concrete patterns for how engineers use AI tools day to day, aligned with Microsoft's Hyper-Velocity Engineering (HVE), including the "velocity is a vector" and "not vibe coding" framings, codifying team standards into AI guidance, writing code for AI agents, cross-functional pairing, and a "getting started" section adapted from HVE's four-step starter with HFE's human-growth lens.
  - Talking points for introduction meetings and audience-specific briefings.
  - Presentable slide deck (Marp-compatible).
  - Developer FAQ.
- GitHub Pages publishing via GitHub Actions, with custom domain support.
